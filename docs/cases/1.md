# Настройка виртуальной валюты. Скачивание балансов сотрудников

Тип: UI  
Приоритет: medium  
Теги: @ui @virtualcurrency  

Стенд: https://client.st7.apprs.ru/  
Роль/данные:
- роль: admin
- креды: `.env.st7`

Предпосылки:
- Админ логинится и открывает https://client.st7.apprs.ru/ru/manager/karma/settings/ (страница настроек виртуальной валюты).

Шаги:
1) На странице настроек виртуальной валюты нажать «Скачать балансы сотрудников».
2) Убедиться, что файл скачан (XLSX), открыть и проверить структуру: столбцы «ФИО / E-mail / Отдел / Валюта для трат в магазине / Валюта для благодарности», все поля заполнены; если сотрудник без отдела — прочерк; если валюты нет — 0.
3) Проверить, что в отчёте присутствуют все сотрудники компании, порядок строк соответствует выгрузке (пример нестандартного порядка допустим).

ОР:
1) По клику на «Скачать балансы сотрудников» получен XLSX, загрузка завершилась без ошибок.
2) Файл читается, структура корректна: столбцы ФИО / E-mail / Отдел / Валюта для трат в магазине / Валюта для благодарности; пустых критичных полей нет; если сотрудник без отдела — прочерк; если валюты нет — 0.
3) В отчёте присутствуют все сотрудники компании, порядок строк соответствует выгрузке (пример нестандартного порядка допустим).

Нюансы:
- Проверяем обязательность всех столбцов и соответствие порядка строк выгрузке (пример с нестандартным порядком, не алфавит).
- Данные берем существующие (ничего не генерируем).
- Открытый вопрос: включать ли архивных/уволенных сотрудников в отчёт?
