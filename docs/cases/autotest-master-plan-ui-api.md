# Актуальный мастер-план автотестов (UI + API + Экспорты)

## Предусловия
- Роли:
  - `admin_full` (полный админ)
  - `review_admin` (админ оценки с ограниченным доступом)
  - `manager` (руководитель)
  - `default_user` (обычный пользователь)
- Набор ревью:
  - `A`, `B` — назначены `review_admin` как администратору оценки
  - `C` — не назначено `review_admin` (недоступная оценка)
  - `LARGE` — ревью с 200+ участниками, назначено `review_admin`
- Данные:
  - Участники приглашены и имеют доступ к аккаунтам
  - Для ревью есть хотя бы одна активная ревизия
  - Ответы заполнены до нужного состояния для сценариев калибровки/экспорта
- Согласованные правила:
  - Если у `review_admin` нет назначенных ревью, на обеих вкладках кнопки выгрузки disabled
  - `Матрица потенциала 9 box` и `Планы развития` для `review_admin` скрыты
  - По прямым ссылкам на скрытые вкладки — стандартная ошибка отсутствия прав
  - Для XLSX/PPTX проверяем состав данных (разрешённые сотрудники/оценки), без строгой сверки агрегатов
